## Progresion docente

1. Primero se explica por que los fundamentos importan en produccion.
2. Luego se baja a mecanismos internos: secuencialidad, self-attention, Q/K/V y bloque Transformer.
3. Despues se conecta tokenizacion con vocabulario, memoria, costo y latencia.
4. Finalmente se introduce adaptacion eficiente con PEFT/LoRA y se integra todo en un ADR financiero.

Cada notebook incluye objetivo, teoria breve, grafico Mermaid, ejemplo Python minimo, interpretacion, errores comunes y mini practica.
